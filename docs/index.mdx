# file-view
基于 React 文件预览组件库。


## 安装

```bash
pnpm add file-view
```

## 依赖

需要安装以下 peer dependencies：

- `react` >= 16.9.0
- `react-dom` >= 16.9.0
- `antd`
- `react-pdf`
- `@ant-design/icons`
- `ahooks`

## 快速开始

```jsx
import { PdfView } from '@luoluo/file-view';

function App() {
  return (
    <div style={{ width: '100%', height: '600px' }}>
      <PdfView
        url="https://example.com/document.pdf"
        fileName="document.pdf"
      />
    </div>
  );
}
```
