# file-view
基于 React 文件预览组件库。


## 安装

```bash
// npm
npm i @luo-luo/file-view

// pnpm
pnpm add @luo-luo/file-view
```

## 依赖

需要安装以下 peer dependencies：

- `react` >= 16.9.0
- `react-dom` >= 16.9.0

## 快速开始

```jsx
import { PdfView } from '@luo-luo/file-view';

function App() {
  return (
    <div style={{ width: '100%', height: '600px' }}>
      <PdfView
        url="https://example.com/document.pdf"
        fileName="document.pdf"
      />
    </div>
  );
}
```
