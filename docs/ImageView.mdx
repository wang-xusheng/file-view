# ImageView

图片预览组件，支持缩放、旋转、下载、全屏等功能。

## 基础用法

传入图片的 URL 即可预览：

```jsx preview
import React from 'react';
import { ImageView } from '@luo-luo/file-view';

export default () => {
    return (
        <div style={{ width: '100%', height: '600px' }}>
            <ImageView
                url="https://raw.githubusercontent.com/wang-xusheng/file-view/refs/heads/feature/photo/example_files/test.avif"
                fileName="test.avif"
                leftDom={<span>test.avif</span>}
            />
        </div>
    )
}
```
